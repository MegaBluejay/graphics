## Dependencies
Tested with PyPy 7.3.9 (3.9.12) and CPython 3.10.7, but should work with any python 3.6+ interpreter

### With pipenv
- `pipenv install`
### With pip
- make a virtualenv (optional)
- `pip install -r requirements.txt`

## Usage
`python main.py`
